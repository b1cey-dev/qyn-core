# Quyn Native Contracts

- **Vesting**: Team allocation 2-year linear vesting (see core/genesis and vesting logic).
- **Faucet**: Testnet faucet contract for devnet/testnet.
- **Staking**: Validator registration and delegation interfaces (native state in consensus crate).

Solidity contracts can be placed here and compiled with solc or Foundry for deployment on Quyn (QVM).
