# Quyn Overview

Quyn (QYN) is a decentralized payment network with smart contract capability. It targets fast, cheap global transfers (like XRP) with EVM compatibility (like Ethereum).

- **Block time**: 3 seconds
- **Consensus**: Proof of Stake (min stake 10,000 QYN, max 1,000 validators, epoch 32 blocks)
- **Supply**: 1 billion QYN, 18 decimals, 0% inflation
- **Gas**: 50% of fees burned, 50% to block proposer
- **Chain ID**: 7777 (mainnet), 7779 (testnet)
